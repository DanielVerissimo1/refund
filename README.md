**Refund** é uma aplicação web para gestão de pedidos de reembolso e listagem de despesas empresariais, permitindo o acompanhamento e a organização das solicitações financeiras.

Este projeto foi desenvolvido durante a formação **FullStack** da **Rocketseat**, aplicando conceitos de front-end e lógica de programação.

## 🚀 Funcionalidades

| Funcionalidade | Descrição |
|----------------|-----------|
| 📝 **Solicitar reembolso** | Adicione novas despesas para reembolso |
| 📋 **Listar despesas** | Visualize todas as solicitações cadastradas |
| ✅ **Aprovar reembolso** | Altere o status da solicitação para aprovado |
| ❌ **Reprovar reembolso** | Altere o status da solicitação para reprovado |
| 🗑️ **Remover solicitação** | Exclua uma despesa da lista |
| 💰 **Cálculo automático** | Some os valores das despesas pendentes |
| 🔄 **Atualização dinâmica** | Interface reage instantaneamente às ações |

## 📚 Conceitos aplicados

- Manipulação do DOM com JavaScript puro (Vanilla JS)
- Gerenciamento de estado com arrays e objetos
- Eventos de clique e submissão de formulários
- Renderização dinâmica de listas
- Armazenamento temporário em memória
- Validação de dados do formulário
- Estilização com CSS moderno (Flexbox/Grid)
